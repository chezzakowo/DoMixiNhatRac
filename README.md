## 🕹️ Nội Dung Game

Giúp anh Độ Mixi dọn rác ở biển bằng lưới, nhưng đừng bắt sinh vật biển nhe!

**Nhiệm vụ của bạn:**
- 🪣 **Giúp anh Độ Mixi** dùng lưới để nhặt rác trôi nổi trên biển
- 🐟 **Tránh bắt nhầm** các sinh vật biển như cá, sao biển và các loài vô tội khác
- 📈 **Gom càng nhiều rác** thì điểm càng cao — hãy lập kỷ lục cho Tộc!

> 🎯 *Mục tiêu: Đạt điểm cao nhất có thể mà không làm hại sinh vật biển nào! Game kéo dài gần như vô hạn (chỉ thua anh Dev Game trùm thế giới nào đó 💔)*

## 📖 Cách Chơi

### 🎮 Điều Khiển

| Phím / Thao Tác | Hành Động |
|-----------------|-----------|
| ⬅️ `←` | Di chuyển sang trái |
| ➡️ `→` | Di chuyển sang phải |
| 🖱️ Di chuột | Điều hướng nhân vật (nếu hỗ trợ) |

### 🎯 Mục Tiêu

1. **Thu gom rác** (chai nhựa, túi nilon, lon rỗng...) bằng lưới
2. **Tránh bắt** cá, sao biển, và các sinh vật biển khác
3. **Ghi điểm cao nhất** bằng khả năng của mình

### ⚠️ Những Điều Cần Tránh

- Bắt sinh vật biển => Trừ 1 tim
- Hết tim thì thua ``¯\_(ツ)_/¯``

## 🛠️ Tùy Chỉnh & Deploy

### 1. Clone repo về máy

```bash
git clone https://github.com/chezzakowo/DoMixiNhatRac.git
cd DoMixiNhatRac
```

### 2. Chạy game trên máy local

Không cần cài đặt gì thêm — chỉ cần mở file `index.html` bằng trình duyệt:

```bash
# Cách 1: Mở thẳng file
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux

# Cách 2: Dùng Live Server (VS Code Extension) để có hot-reload
# Cài extension Live Server → Click chuột phải vào index.html → "Open with Live Server"
```

### 3. Chỉnh sửa Asset

Thay thế hoặc chỉnh sửa các file sau để tuỳ biến nhân vật và đồ vật:

```
image.jpg     ← Thay ảnh nhân vật / background
music.mp3     ← Thay nhạc nền trong game
menu.mp3      ← Thay nhạc màn hình menu
```

> 🎨 Asset gốc được tham khảo từ project Scratch: [scratch.mit.edu/projects/759277620](https://scratch.mit.edu/projects/759277620/)

### 4. Chỉnh sửa Logic Game

Mở `game.js` và tìm các biến sau để điều chỉnh gameplay:

```javascript
// Ví dụ các thông số có thể chỉnh trong game.js:
const SCORE_TRASH   = 10;   // Điểm khi nhặt rác
const SCORE_PENALTY = -15;  // Điểm bị trừ khi bắt nhầm sinh vật
const GAME_TIME     = 60;   // Thời gian chơi (giây)
const SPAWN_RATE    = 1.5;  // Tốc độ xuất hiện vật thể
```

### 5. Deploy lên GitHub Pages

```bash
# Bước 1: Đảm bảo repo đã có trên GitHub
git add .
git commit -m "Update game"
git push origin main

# Bước 2: Vào Settings → Pages → Source: Deploy from branch → main → / (root)
# Bước 3: Truy cập game tại:
# https://<username>.github.io/DoMixiNhatRac/
```

Hoặc deploy nhanh với **Netlify** bằng cách kéo thả thư mục repo vào [app.netlify.com](https://app.netlify.com/drop) — xong ngay trong 10 giây! ⚡
